Line Chart

Source: Lexicon Charts — Line chart. Multi-series SVG line chart. Every series uses one hue (--chart-2, blue); they're told apart by marker shape (circle / square / triangle / diamond) and line style (--dashed / --dotted). Colourblind-safe: it never relies on colour alone. Add .chart-c1….chart-c4 to the series groups for a multi-colour version. The legend repeats the marker and prints the last value in bold.
